# Astronomy

This library is used for various astronomical calculations, procedural generation algorithms, and other related tasks.

## Features

- World generation, from an astronomical perspective:
  - size of the planet
  - solar system configuration
  - moons
  - atmosphere
  - length of day
  - axial tilt
  - orbital eccentricity
  - etc
- Astronomical calculations:
  - sunrise/sunset
  - moonrise/moonset
  - moon phase
  - solar eclipse
  - lunar eclipse
  - etc
- Procedural generation:
  - star maps
  - planets
  - moons
  - constellations
  - etc
