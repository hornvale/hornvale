# The Lode Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Turn The Ground's `cave_proneness`/`prospectivity` probability *fields* into located, typed **caves** and **ore deposits** that live in The Deep's column — a pure, deterministic, no-epoch derived layer extending `domains/terrain`.

**Architecture:** A new `domains/terrain/src/features.rs` of pure classifiers, driven by a deterministic hash-noise point process in the `GeneratedTerrain` provider (the `cave_proneness_at`/`column_at` pattern). Caves and deposits share one fluid-flow gate (so they co-locate) and bias along tectonic lineaments (so ore forms belts). The one climate-coupled deposit (laterite) is wired at the worldgen composition root, like The Ground's soil. Mundane ores only; magical ores stay reserved.

**Tech Stack:** Rust (edition 2024), `hornvale-terrain` (the shipped `MaterialBuffer`, `RockClass`, `BoundaryKind`, `strata::BandKind`, `column_at`), kernel noise (`SphereFbm`/`sphere_fbm01`), `hornvale-worldgen` (the `soil_of` climate-coupling seam), `hornvale-lab`, `hornvale-almanac`. No new dependencies.

## Global Constraints

- **No epoch / byte-identity:** no committed byte of any existing world changes. `cli/tests/lens_purity.rs` stays green. Do **not** rebaseline `cli/tests/fixtures/world-seed-42.json`.
- **No new committed facts.** Features are a non-serialized derived query; committing a fact would move the golden.
- **Zero new *sequential* stream draws.** One new *hash-noise* label `FEATURES` (derived via `terrain_seed.derive(streams::FEATURES)`, sampled through `SphereFbm`, never `.stream()`) — additive/safe, no consumption-order contract, exactly The Ground's `LITHOLOGY` mechanism.
- **Narration, never integration.** Pure functions of the genesis-time globe; no forward integrator.
- **Determinism:** same seed + pins → byte-identical features & census rows. No `HashMap`/`HashSet`; float sorts `total_cmp`; no wall-clock.
- **Mundane ores only; magical ores reserved** (The Ground §8) — the taxonomy stops at gold/gems; no magical member ships.
- **Typed quantities + type-audit:** every new `pub`-boundary primitive carries a `type-audit:` verdict tag (physical/ratio readouts `bare-ok(ratio)` or `bare-ok(diagnostic-value)`; counts `bare-ok(count)`; bools `bare-ok(flag)`; enums need no tag). CI is default-deny.
- **Docs:** `#![warn(missing_docs)]` — every pub item/field/variant documented. `cargo fmt` is the final step before committing.
- **Land-only; one dominant deposit per cell; deposits are neutral truth** (cultural knowledge deferred).

**Verification commands:** scoped `cargo test -p hornvale-terrain <name>`; byte-identity `cargo test -p hornvale --test lens_purity`; type-audit `cargo run --manifest-path tools/type-audit/Cargo.toml -- check`; commit gate `make gate`.

```rust
use crate::lithology::MaterialBuffer;

/// A cave type, by the lithologic process that opened the void.
#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum CaveKind {
    /// Carbonate dissolution (wet limestone).
    Karst,
    /// Drained basaltic/volcanic tube.
    LavaTube,
    /// Fault/fracture void in tectonically active rock.
    Fracture,
}

/// A located cave at a cell.
/// type-audit: bare-ok(count: depth_reach_bands)
#[derive(Debug, Clone, Copy, PartialEq)]
pub struct Cave {
    /// Which process opened it.
    pub kind: CaveKind,
    /// How many of the column's bands the void penetrates (1 = cover only).
    pub depth_reach_bands: u32,
}

/// The cave type implied by lithology: carbonate dissolves to karst, mafic
/// (low-silica) volcanic drains to lava-tube, else a fault fracture.
pub fn cave_kind(buf: &MaterialBuffer, near_fault: bool) -> CaveKind {
    if buf.carbonate > 0.5 {
        CaveKind::Karst
    } else if buf.silica < 0.3 {
        CaveKind::LavaTube
    } else {
        let _ = near_fault; // fracture is the fallback; near_fault reinforces presence, not kind
        CaveKind::Fracture
    }
}

/// Lineament proximity weight: features cluster into belts near plate contacts.
/// `hops` is boundary distance (fewer = closer); `None` = cratonic interior.
/// type-audit: bare-ok(count: return)  <!-- returns a ratio; see note -->
pub fn belt_weight(hops: Option<u32>) -> f64 {
    match hops {
        Some(h) => 1.0 / (1.0 + h as f64 * 0.5),
        None => 0.3,
    }
}

/// Belt-weighted presence probability for a feature, from its gating field.
/// type-audit: bare-ok(ratio: field), bare-ok(ratio: belt), bare-ok(ratio: return)
pub fn presence_prob(field: f64, belt: f64) -> f64 {
    (field * (0.4 + 0.6 * belt)).clamp(0.0, 1.0)
}
```

```rust
use crate::RockClass;
use crate::boundaries::BoundaryKind;
use crate::strata::BandKind;

/// The genetic process that formed a deposit — the taxonomy's primary axis.
#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum DepositProcess {
    /// Arc/intrusion magmatic (porphyry).
    Magmatic,
    /// Fault/orogen hydrothermal vein.
    Hydrothermal,
    /// Carbonate-hosted (MVT).
    CarbonateHosted,
    /// Chemical/biogenic sediment — the areal bedded ores.
    ChemicalSediment,
    /// Felsic-intrusion pegmatite (and metamorphic gems).
    Pegmatite,
    /// Secondary placer (eroded, in alluvium).
    Placer,
    /// Residual laterite (hot+wet weathering) — wired at worldgen (climate).
    Lateritic,
}

/// The commodity a deposit yields.
#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum Commodity {
    /// Copper (magmatic arc).
    Copper,
    /// Gold (hydrothermal vein or placer).
    Gold,
    /// Lead-zinc (carbonate-hosted).
    LeadZinc,
    /// Iron (BIF/ironstone).
    Iron,
    /// Salt/potash (evaporite).
    Salt,
    /// Coal (biogenic sediment).
    Coal,
    /// Gems + rare metals (pegmatite/metamorphic).
    Gems,
    /// Tin (placer).
    Tin,
    /// Bauxite/nickel (laterite).
    Bauxite,
}

/// A located ore deposit.
/// type-audit: bare-ok(ratio: grade), bare-ok(ratio: tonnage)
#[derive(Debug, Clone, Copy, PartialEq)]
pub struct Deposit {
    /// Its genetic process.
    pub process: DepositProcess,
    /// Its commodity.
    pub commodity: Commodity,
    /// Which column band it sits in.
    pub depth: BandKind,
    /// Ore grade [0,1], lognormal — many poor, few rich.
    pub grade: f64,
    /// Relative tonnage [0,1]: small rich veins → huge disseminated bodies.
    pub tonnage: f64,
}

/// The dominant deposit family for a cell, from rock + tectonic setting.
/// Areal ores (iron/salt/coal) project directly from the rock class; the point
/// ores read the setting. Returns `None` where nothing is prospective.
pub fn deposit_kind(
    rock: RockClass,
    boundary: Option<BoundaryKind>,
    buf: &MaterialBuffer,
    endorheic: bool,
) -> Option<(DepositProcess, Commodity)> {
    use Commodity::*;
    use DepositProcess::*;
    // Areal bedded ores: the rock IS the ore.
    match rock {
        RockClass::Ironstone => return Some((ChemicalSediment, Iron)),
        RockClass::Evaporite => return Some((ChemicalSediment, Salt)),
        RockClass::Coal => return Some((ChemicalSediment, Coal)),
        RockClass::Alluvium => return Some((Placer, Gold)),
        _ => {}
    }
    if endorheic && buf.carbonate < 0.2 {
        return Some((ChemicalSediment, Salt));
    }
    // Point/vein/belt ores by setting.
    if buf.carbonate > 0.5 {
        return Some((CarbonateHosted, LeadZinc));
    }
    if buf.silica > 0.6 && buf.grain > 0.5 {
        return Some((Pegmatite, Gems)); // felsic + coarse (plutonic)
    }
    match boundary.map(setting_of) {
        Some(Setting::Arc) => Some((Magmatic, Copper)),
        Some(Setting::Fault) => Some((Hydrothermal, Gold)),
        _ => None,
    }
}

/// Coarse tectonic setting for deposit typing.
enum Setting {
    Arc,
    Fault,
    Other,
}
/// Map a boundary kind to a coarse setting. (Match the real `BoundaryKind`
/// variants: convergent/subduction → Arc; transform/divergent → Fault.)
fn setting_of(kind: BoundaryKind) -> Setting {
    match kind {
        // ADAPT to the real BoundaryKind variant names:
        BoundaryKind::Convergent => Setting::Arc,
        BoundaryKind::Transform | BoundaryKind::Divergent => Setting::Fault,
        _ => Setting::Other,
    }
}

/// The column band a deposit's process places it in (primary deep, secondary shallow).
pub fn deposit_depth(process: DepositProcess) -> BandKind {
    match process {
        DepositProcess::Placer | DepositProcess::Lateritic => BandKind::Regolith,
        DepositProcess::ChemicalSediment | DepositProcess::CarbonateHosted => BandKind::Cover,
        DepositProcess::Magmatic => BandKind::Roots,
        DepositProcess::Hydrothermal | DepositProcess::Pegmatite => BandKind::Basement,
    }
}

/// Grade (lognormal-shaped: many poor, few rich) and relative tonnage from the
/// process and the gating prospectivity, using a hash-noise value in `[0,1)`.
/// type-audit: bare-ok(ratio: prospectivity), bare-ok(ratio: noise)
pub fn deposit_grade_tonnage(process: DepositProcess, prospectivity: f64, noise: f64) -> (f64, f64) {
    // Lognormal-ish: cube the uniform noise so most grades are low, a few high.
    let grade = (prospectivity * noise * noise * noise * 4.0).clamp(0.0, 1.0);
    // Disseminated magmatic = huge tonnage/low grade; veins = small/rich.
    let tonnage = match process {
        DepositProcess::Magmatic | DepositProcess::ChemicalSediment => 0.7 + 0.3 * noise,
        DepositProcess::Hydrothermal | DepositProcess::Pegmatite => 0.1 + 0.3 * noise,
        _ => 0.3 + 0.4 * noise,
    };
    (grade, tonnage.clamp(0.0, 1.0))
}
```

```rust
/// The canonical deposit query: terrain's derived deposits, overlaid with the
/// one climate-coupled ore (laterite — bauxite/nickel from hot+wet weathering)
/// where terrain has no stronger primary. The soil-projection seam (The Ground)
/// applied to ore. (The Lode, spec §4.)
pub fn deposit_of(
    terrain: &GeneratedTerrain,
    climate: &GeneratedClimate,
    geo: &Geosphere,
    cell: hornvale_kernel::CellId,
) -> Option<hornvale_terrain::features::Deposit> {
    if let Some(d) = terrain.deposit_at(cell) {
        return Some(d); // a primary/areal deposit wins
    }
    if terrain.is_ocean(cell) {
        return None;
    }
    let _ = geo;
    let hot = climate.mean_temperature_at(cell).get() > 22.0;
    let wet = climate.moisture_at(cell) > 0.6;
    if hot && wet {
        return Some(hornvale_terrain::features::Deposit {
            process: hornvale_terrain::features::DepositProcess::Lateritic,
            commodity: hornvale_terrain::features::Commodity::Bauxite,
            depth: hornvale_terrain::strata::BandKind::Regolith,
            grade: 0.4,
            tonnage: 0.6,
        });
    }
    None
}
```
