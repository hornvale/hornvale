# The Haunt ‚Äî Implementation Plan

**Status: COMPLETE (shipped 2026-07-22).** T1 `b43dfedb`, T2 `300b8439` + gallery
`ec79bb07`, T3 `519a0cba`. The byte-identity premise was corrected mid-execution
(decision-ledger #4, Nathan-approved): wild fauna DO get frightened, so success is
scoped drift (only possession-over-time moves), not byte-identity; the penalty was
tuned to 5 to avoid the Dijkstra-budget plan-failure.


Remembered danger (PSY-11's reserved instance). Spec:
`docs/superpowers/specs/2026-07-22-the-haunt-design.md`. A `believed_hazard` fold
(the inverted twin of `believed_water`) ‚Äî the set of cells a creature was
frightened at ‚Äî read as a **finite route cost** in the shared planner, so its
thirst/homing paths bend AROUND shunned ground. The Danger flow drive is
unchanged; memory lives at the planning layer. Derived, no epoch; empty set on
seed 42 ‚áí byte-identical.

**Goal:** A creature frightened at a place plans its later journeys to water and
home to detour around it ‚Äî proactively ‚Äî yet still reaches water when the detour
would exceed the penalty (braves it), while every current world stays byte-
identical.

## Global Constraints

- No `HashMap`/`HashSet` ‚Äî `BTreeMap`/`BTreeSet`/`Vec` only. Float order via
  `f64::total_cmp` with deterministic tie-breaks; A* costs are `u64`.
- No wall-clock time. `f64` transcendentals via `hornvale_kernel::math`.
- **Byte-identity:** `new --seed 42` and the seed-42 possession galleries are
  byte-for-byte unchanged (the remembered set is empty ‚Äî the settled peoples are
  never frightened, so every planner edge cost stays `1`). The gate's artifact-
  drift check enforces it.
- **Determinism:** `believed_hazard` is a fold over already-committed `agent-at`
  facts + terrain danger ‚Äî no seed draw, no new predicate, **no epoch**. The
  `BTreeSet` is ordered; the A* over `u64` costs is deterministic with the
  existing tie-breaks.
- Every crate `#![warn(missing_docs)]`; every pub item documented. type-audit tag
  every pub-boundary primitive; run the type-audit check before each commit.
- Dependencies: `serde` + `serde_json` only. All work in `windows/vessel`
  (+ a lab null-control check in Task 3). No worldgen/domain change.
- `cargo fmt` final before every commit. Iterate cost-ordered; `make gate` is the
  final pre-commit step for boundary changes.
- Test terrains use the `Default`-based `PlantedTerrain`/`SyntheticTerrain`
  builders (a new field is one struct-field default).

### Task 1: The `believed_hazard` fold

The memory, self-contained and testable over a planted `agent-at` history +
planted terrain danger. Implements spec ¬ß1.

**Files:**
- Modify: `windows/vessel/src/liveness.rs` (the `believed_hazard` fn; a
  `frightened_at` helper)
- Test: `windows/vessel/src/liveness.rs` test module (beside the `believed_water`
  tests)

**Interfaces:**
- `pub fn believed_hazard(ledger: &Ledger, npc: &Npc, t: WorldTime, terrain: &dyn Terrain) -> std::collections::BTreeSet<RoomAddr>`
  ‚Äî mirrors `believed_water` (liveness.rs:817): fold the creature's committed
  `agent-at` history (`f.subject == npc.entity`, `day ‚â§ t`), and collect each
  visited cell that is **frightened-at** for this creature. Returns the set (empty
  when never frightened). Pure over the frozen ledger.
- A `frightened_at(room, npc, terrain) -> bool` helper: the creature's felt threat
  at `room` crosses `DANGER_ACT` ‚Äî i.e. reuse the Danger drive's own reading:
  `max over room+neighbours of threat_value(npc.threat_niche, terrain.hazards(¬∑))`,
  scaled by `mettle_factor(npc.boldness)`, `‚â• DANGER_ACT`. (This is exactly the
  Danger drive's `urgency` computation; factor it so drive and memory agree ‚Äî one
  source of truth, no drift.) For v1's static terrain this equals "still dangerous
  there," so the most-recent-visit staleness rule (spec ¬ß1) reduces to
  "visited ‚àß frightened-now" and needs no visit-order bookkeeping ‚Äî document that.
- type-audit: `BTreeSet<RoomAddr>` return (typed keys, no bare primitive) ‚Äî no new
  tag; `frightened_at` returns `bool` ‚Üí `bare-ok(flag: return)` if pub (keep it
  private if possible ‚Üí no tag).

**Steps (TDD):**

  - `believed_hazard_is_empty_when_never_frightened`: a history over hazard-free

  - `believed_hazard_holds_the_visited_dangerous_cells`: a history visiting a
    planted UNCANNY-hazard cell (Danger crosses `act`) and some safe cells ‚Üí the
