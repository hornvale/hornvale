# 0017. Campaigns drop the "Year" naming

**Status:** Accepted (2026-07-07) · **Decider:** Nathan

In the context of campaign planning having outgrown the Year-N framing,
facing terminology that bears no relation to calendar time and actively
misleads, we decided that **future campaigns are identified by a plain
sequence number plus a name** ("Campaign 14: The Peoples"), accepting a
mixed historical record.

**Context.** "Year 1" and "Year 2" were pacing scaffolding from the
long-term plan. The project moves far faster than the framing implies, so
the prefix now carries misinformation instead of meaning. The names ("The
Sky," "The Peoples") were already doing the semantic work; the numbers
only need to sort.

**Consequence.** The convention changes at the next campaign boundary,
forward-only. Existing Y2-0/Y2-1 specs, plans, branches, and chronicle
entries keep their names — the same discipline as the epoch-suffix rule
and this log's never-edit rule. The chronicle remains the authoritative
campaign order. If thematic grouping is wanted later, named **arcs** carry
it without pretending to be calendar time.

**See also.** The Year-2 metaplan spec (last document of the old naming);
decision 0006 (labels are permanent contracts — the same instinct applied
to process history).
