# The Corrigendum

*A priesthood taught the darkening would come on day 36,528. It came on
day 36,522. The Book says so, in the margin, where it says everything the
account above couldn't.*

Every prediction this project has rendered so far has been true, because
every prediction has secretly been a lookup: a priesthood's taught day was
the real future's own answer, read off the closed-form ephemeris and
handed to the culture as if it had earned it. This campaign takes that
back. A priesthood now teaches only what it could actually have
extrapolated — the last darkening it witnessed, plus the average gap
between every darkening before that — and the real sky owes that
extrapolation nothing. An eclipse's true timing beats gently against the
calendar year, the same mechanism behind the real saros cycle, and a
naive average doesn't know to expect it. Most of the time the difference
is too small to matter. Sometimes it isn't, and two wrong guesses in a
row — never just one, real revision doesn't fire on an anomaly — is a
live crisis, printed in the Book's own margin beside the priesthood's
confident, present-tense teaching: what was taught, and what came
instead.

Nothing had to be invented to make the crisis heal. A knowledge-state in
this project has always been a fresh derivation from everything a culture
has witnessed by that day — never a stored belief waiting to be revised,
because there has never been anything to revise *to*. A later day simply
witnesses more, and the same extrapolation, run again over more evidence,
teaches better. Revision was already the whole shape of how this project
knows anything; this campaign is the first time getting something wrong
made that shape visible.

An organized priesthood says one thing more than a folk memory does,
speaking in its own doctrinal voice: that no one has caught its teaching
false, or — once someone plausibly could have — that they did. The
sentence echoes, deliberately, the exact shape The Doctrine already built
for a verifiable claim an institution stakes its authority on: the cell
that campaign named the Galileo cell, real but never yet driven by an
actual falsified claim. This campaign gives it its first one, in prose
only — the doctrine line is thematic, not routed through that machinery's
own conflict map, a real second campaign's work left named rather than
built.
